### Виджеты

Добавлен новый виджет «AI-чат» — для предоставления пользователям преднастроенных вопросов для взаимодействия с AI-провайдерами. Подробнее — в разделе [«AI-чат»](../../admin/widgets/types/#ai-чат).

### Каталог

На странице ресурса теперь можно переключать отображение списка сущностей между таблицей и карточками. Подробнее — в разделе [«Режим просмотра сущностей»](../../user/catalog/#режим-просмотра-сущностей).
