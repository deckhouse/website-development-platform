## Новые возможности

### Фильтрация групп при синхронизации

Добавлена настройка правил фильтрации групп при синхронизации команд ([подробнее](../../user/teams-users/#фильтрация-групп-при-синхронизации)).

- Правила применяются по порядку цепочкой: каждое правило фильтрует результат предыдущего.
- Поддерживаются два режима:
  - **Включить** — синхронизировать только команды, совпадающие с паттерном.
  - **Исключить** — синхронизировать только команды, не совпадающие с паттерном.
- Паттерны задаются в виде регулярных выражений.
- Настройка доступна в разделе «Администрирование» → «Команды» по кнопке «Правила синхронизации».

### Ролевая модель

- Добавлено глобальное разрешение `edit:team-filter-rules` для настройки правил фильтрации групп при синхронизации ([подробнее](../../admin/security/rbac/#глобальные-разрешения)).
