Интерфейс Deckhouse Development Platform включает в себя следующие разделы:

- **Главная** — стартовая страница. На неё могут быть прикреплены дашборды с виджетами.
- **Каталог** — сервисный каталог, раздел для отображения ресурсов, сущностей, связей, а также для запуска действий и сценариев.
- **Самообслуживание** — раздел для настройки источников данных, действий, вебхуков, автоматизаций, сценариев, дашбордов и виджетов. Доступ к разделу может быть ограничен ролевой моделью.
- **Администрирование** — раздел для управления командами, пользователями, политиками доступа и учётными данными. Доступ к разделу может быть ограничен ролевой моделью.

## Глобальный поиск

В верхней панели интерфейса платформы доступна кнопка **Поиск**, которая позволяет быстро находить сущности по всей платформе.

### Использование поиска

1. **Открытие поиска** — нажмите на кнопку "Поиск" в верхней панели интерфейса.
2. **Ввод запроса** — начните вводить текст в поле поиска.
3. **Просмотр результатов** — результаты поиска отображаются в выпадающем списке под полем ввода. Для каждой найденной сущности отображается:
   - Название сущности.
   - Идентификатор сущности.
   - Идентификатор (slug) сущности.
   - Ресурс, в котором находится сущность.

### Ограничения

- Максимальная длина поискового запроса — 255 символов.
- Поиск выполняется только среди сущностей, созданных в платформе. Ресурсы, команды и другие объекты платформы не участвуют в поиске.
